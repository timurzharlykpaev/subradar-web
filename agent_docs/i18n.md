# Интернационализация (i18n)

## Стек

**react-i18next** + **i18next-browser-languagedetector**

Конфигурация: `src/lib/i18n.ts`

---

| `de` | Deutsch | `src/locales/de.json` |

---

## Использование в компонентах

```tsx
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();

  return <h1>{t('dashboard.title')}</h1>;
}
```

## Правила работы с переводами

### Добавление нового ключа
1. Добавь ключ в `en.json` (основной)
2. Добавь тот же ключ во **все** остальные 8 файлов локализации
3. Если нет перевода — используй английское значение как fallback

### Изменение существующего ключа
1. Измени во **всех** файлах локализации одновременно
2. Найди все использования ключа в компонентах через поиск

### Удаление ключа
1. Удали из **всех** файлов локализации
2. Удали все `t('ключ')` в компонентах

## Переключение языка

Программное переключение:
```ts
import i18n from '@/lib/i18n';

i18n.changeLanguage('ru');
localStorage.setItem('subradar-language', 'ru');
```

Реализовано в `SettingsPage` через `handleLanguageChange()`.

Детекция языка при запуске:
1. Читает `localStorage.getItem('subradar-language')`
2. Если нет — читает `navigator.language`
3. Fallback: `en`

## Настройки пользователя

Язык сохраняется в двух местах:
- `localStorage` (`subradar-language`) — для i18next detector
- `useAppStore().language` — в Zustand store
- `PATCH /users/me { locale }` — синхронизируется с бэкендом при сохранении профиля
